- A（通用回归通过率）：

红线条件（任一触发即 `不通过`）：

- `S2/H5/E3/P4/X4` 任一失败
- 默认入口非 `steward`
- `C11/C05/C13` 未完成验证却进入默认命中
- 结论缺少核心证据链

结论口径：

- 通过：
  - 红线 0 触发
  - A >= 90
  - B = 100
  - C >= 95
  - D = 100
  - 总分 >= 90
  - 含义：可进入 v2 受控试运行
- 有条件通过：
  - 红线 0 触发
  - A >= 80
  - C >= 95
  - D >= 95
  - 总分 >= 80
  - 含义：可进入整改复测或正式评审下一轮，不可进入 v2 受控试运行
- 不通过：
  - 任一红线触发，或 A < 80，或 C < 95，或 D < 95
  - 含义：禁止进入下一阶段，必要时回滚
