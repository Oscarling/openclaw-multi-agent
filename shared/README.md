# Shared Assets

这个目录用于放多 Agent 共用的素材，不放单个角色专属规则。

## 子目录

- `briefs/`：需求简报、选题输入、实验任务卡
- `templates/`：选题卡模板、发布包模板、复盘模板
- `outputs/`：跨角色共享的产出样例
- `skills/`：后续若有共用 SOP、提示词片段或技能说明，可放这里

## 当前原则

- 共享目录只放共用素材
- 角色边界、身份、规则放到 `roles/` 或容器内角色专属目录
